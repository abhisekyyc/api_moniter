# api_moniter
API Performance Monitor using Flask
“Built an API Performance Monitoring Web App using Flask and SQLite that tracks response times and logs API performance.”
Real-time monitoring
Database integration
Web dashboard

A simple web application built using Flask that monitors API response times.

## 🔧 Features
- Check API response time
- Store logs in SQLite database
- Simple web interface

## 🛠️ Tech Stack
- Python
- Flask
- SQLite

## ▶️ How to Run
1. Install dependencies:
   pip install flask requests

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## 📸 Output
Enter API URL and view response time logs instantly.
